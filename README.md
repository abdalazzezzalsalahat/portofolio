# Abdalaziz Asalhat | DevOps Engineer Portfolio

A premium, modern, and interactive portfolio website built for **Abdalaziz Ali Asalhat**. This project showcases a unique blend of professional DevOps expertise and personal flair through a dynamic "Formal vs. Casual" theme switcher.

## ✨ Key Features

- **Dual-Mode Experience**: Switch between **Formal** and **Casual** personas with a single toggle. 
    - **Formal Mode**: Deep blue "Nebula" aesthetic focused on professional experience and technical cloud expertise.
    - **Casual Mode**: Vibrant "Neon Tech" aesthetic highlighting personal interests, hobbies, and social persona.
- **Dynamic Color Themes**: Accents, gradients, and UI elements smoothly transition between "Trust Blue" and "Neon Pink" based on the selected mode.
- **Glassmorphism Design**: Modern UI using frosted glass effects for a high-end, futuristic feel.
- **Animated Anime Stickers**: Interactive floating stickers in the "Beyond the Code" section that add personality and depth.
- **DevIcon Integration**: Visual representation of the technical stack (AWS, Kubernetes, Terraform, Python, Docker, etc.).
- **Fully Responsive**: Optimized for seamless viewing on desktop, tablet, and mobile devices.

## 🛠️ Tech Stack

- **HTML5**: Semantic structure and modern SEO practices.
- **CSS3 (Vanilla)**: Custom styling system with CSS Variables for theme management, Flexbox/Grid layouts, and CSS Keyframe animations.
- **JavaScript**: Smooth scroll reveal effects and interactive navbar behavior.
- **Assets**: High-resolution custom backgrounds and hand-crafted anime-style stickers.

## 🚀 Getting Started

Simply open the `index.html` file in any modern web browser to view the portfolio.

```bash
# To run a simple local server (requires Python)
python3 -m http.server 8000
```
Then navigate to `http://localhost:8000`.

## 📂 Project Structure

- `index.html`: Main content and structure.
- `style.css`: Design system and global theme logic.
- `script.js`: Interactive animations.
- `assets/`: Background images, profile photos, and stickers.
